# 婚戀工具集 💕

台灣單身男女必玩的互動工具合集

## 📋 專案介紹

這是一個結合統計學與婚戀市場分析的交互式工具集，包含：

### 1. 📊 PR 值計算機
基於常態分佈與台灣婚戀市場數據的個人評分系統。
- **分析維度**：年齡、年收入、身高、體重（BMI）、學歷、職業、家世
- **統計方法**：Log-Normal 分佈（收入）、Gaussian Distribution（身高等）
- **輸出結果**：個人 PR 值、雷達圖、詳細分析表

### 2. 💬 女友送命題
4 大題庫涵蓋情侶互動、性格相容、愛情觀與未來規劃。
- **情侶題**：第一次約會、生日禮物、欣賞之處
- **性格題**：性格類型、衝突應對、休閒方式
- **愛情觀**：愛情要素、長期期許、能接受的底線
- **未來規劃**：五年目標、生活方式、育兒想法

## 📊 計算模型

### PR 值權重分配
| 維度 | 權重 |
|------|------|
| 年收入 | 20% |
| 職業 | 15% |
| 年齡優勢 | 15% |
| 身高 | 15% |
| 家世 | 15% |
| 學歷 | 10% |
| 體態（BMI） | 10% |

### 統計分佈
- **收入**：Log-Normal Distribution（中位數 56 萬、PR90 約 120 萬）
- **身高**：High Normal Distribution
  - 男性：平均 173.5 cm，標準差 5.8 cm
  - 女性：平均 160.0 cm，標準差 5.0 cm
- **年齡**：Reverse Normal Distribution（越年輕越優勢）
- **BMI**：高斯衰減函數（理想值 22，容許度 ±3.5）

## 📝 使用說明

### PR 計算機
1. 選擇性別
2. 依序輸入個人資料
3. 點擊「⚡ 執行複雜運算」
4. 查看結果：
   - 綜合 PR 值大卡
   - 7 維度雷達圖
   - 詳細分析表格

### 女友送命題
1. 選擇題庫分類（情侶題/性格題/愛情觀/未來規劃）
2. 逐題選擇答案
3. 用「上一題」「下一題」導航
4. 完成後顯示完成提示
