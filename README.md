# 婚戀工具集 💕

台灣單身男女必玩的互動工具合集

## 📋 專案介紹

這是一個結合統計學與婚戀市場分析的交互式工具集，包含：

### 1. 📊 PR 值計算機
基於常態分佈與台灣婚戀市場數據的個人評分系統。
- **分析維度**：年齡、年收入、身高、體重（BMI）、學歷、職業、家世
- **統計方法**：Log-Normal 分佈（收入）、Gaussian Distribution（身高等）
- **輸出結果**：個人 PR 值、雷達圖、詳細分析表

### 2. 💬 親密關係經營力測驗
新版量表「親密關係經營力（Intimate Relationship Connection Scale）」。
- **題型**：6 題單頁式量表（互動題、陪伴題、生活節奏、延伸反應、愛情觀、未來規劃）
- **作答體驗**：點整個選項即可選取，系統背景即時計分
- **結果機制**：按送出後自動跳轉到結果區，顯示主類型、次要傾向、A-E 分數與進階判讀
- **分類邏輯**：A 穩定共創型、B 高感連結型、C 穩定平衡型、D 低頻獨立型、E 流動觀察型

### 3. 🎮 送命題挑戰遊戲
即時問答生存遊戲，依人格模式改變同一題的得分邏輯。
- **開局流程**：先選擇人格模式（務實獨立型 / 浪漫感性型 / 隨和鬆弛型）
- **節奏機制**：開局首題等待 3 秒，之後題目間固定等待 1 秒
- **倒數與分數**：每題 5 秒倒數，逾時扣分；分數歸零即結束
- **題目追蹤**：顯示已測題數與未測題數

1. 開啟 `index.html` 作為主頁入口

   - 「開始測驗」 → `girlfriend-questions.html`

## 📊 計算模型

### PR 值權重分配
| 維度 | 權重 |
|------|------|
| 年收入 | 20% |
| 職業 | 15% |
| 年齡優勢 | 15% |
| 身高 | 15% |
| 家世 | 15% |
| 學歷 | 10% |
| 體態（BMI） | 10% |

### 統計分佈
- **收入**：Log-Normal Distribution（中位數 56 萬、PR90 約 120 萬）
- **身高**：High Normal Distribution
  - 男性：平均 173.5 cm，標準差 5.8 cm
  - 女性：平均 160.0 cm，標準差 5.0 cm
- **年齡**：Reverse Normal Distribution（越年輕越優勢）
- **BMI**：高斯衰減函數（理想值 22，容許度 ±3.5）

## 📝 使用說明

### PR 計算機
1. 選擇性別
2. 依序輸入個人資料
3. 點擊「⚡ 執行複雜運算」
4. 查看結果：
   - 綜合 PR 值大卡
   - 7 維度雷達圖
   - 詳細分析表格

### 親密關係經營力測驗
1. 完成 6 題作答
2. 每次選擇都會背景即時計分
3. 點擊「送出並查看結果」
4. 自動跳轉到結果區，查看主類型、次要傾向與詳細分析

### 送命題挑戰遊戲
1. 開局先選人格模式
2. 題目出現後在倒數時間內選擇回應
3. 首題等待 3 秒，之後每題間隔 1 秒
4. 分數歸零時遊戲結束，可重新挑戰
